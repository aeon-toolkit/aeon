# Contributing to `aeon`

`aeon` is a community-driven project and high-quality contributions are most welcome.
We value all kinds of contributions, not just code. Improvements to docs, bug reports,
and taking on communications or code of conduct responsibilities are all examples of
valuable contributions beyond code which help make `aeon` a great package.
